# 레미콘 프로젝트

레미콘 차량의 DTG 데이터 연동·운행 점수 산출 프로젝트. **아이나비 의존성 제거** 및 **자체 웹 서비스 대체**가 핵심.

## 진척

| 일자 | 진척 |
|------|------|
| 2026-03-30 | DTG 레미콘 프로젝트 신고서 및 사업 계획서 작성 중. |
| 2026-04-06 | 사업계획서 작성 마치고 금일 중 신고 접수 완료. |
| 2026-04-13 | 지넥스 측에 위치정보사업자 등록 재요청. |
| 2026-04-20 | DTG 레미콘 프로젝트 **데이터 연동 API 개발 완료, 스트레스 테스트 예정**. |
| 2026-04-27 | 지넥스 테스트 환경 정보 전달 완료 → **아이나비 의존성 제거 가능성 확인**. |
| 2026-05-04 | 데이터 전송 테스트 완료. 스트레스 테스트를 위한 수량/건수 요청 대기. **아이나비 호출 없이 자체 웹 서비스 대체 방안 검토**. |
