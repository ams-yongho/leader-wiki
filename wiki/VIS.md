# VIS (Vehicle Information Service / "더 낸 보험료 찾기")

자동차 안전관리 특약 누락으로 인한 보험료 과납을 추출·검증해 보험사 또는 차주에게 제공하는 서비스. **2026년 핵심 사업 트랙** 중 하나.

## 비즈니스 모델 진화

### Phase 1: B2C "더 낸 보험료 찾기" (초기)

- 차주에게 직접 과납 보험료 환급 서비스 제공, 수수료 수취 모델.

### Phase 2: 보험사 직판 (2026-04 ~)

- **2026-04-13** 보험사들이 과납 보험료 발생 사실에 부담 → 손해보험협회 미팅 후 [[DB손해보험]]·[[KB손해보험]] 측 회신. **B2C 수수료 모델 → 보험사 직접 데이터 판매**로 전환 검토. 경쟁사 등장 전 가격 협상력 확보.
- 비용/수익성: 보험사 연간 피해액 최대 **2,000억 원** 추산. 2,500만 대 데이터 콜 비용(원가 약 1,600억 원)을 보험사 청구. DE(데이터 처리)에 2,500만 건 일괄 위탁 비용 협상(약 10억 원).
- **2026-04-20** 보험사들과 각 세우지 않고 데이터 제값 판매 우선. 원활하지 않을 경우 2026.06 [[혁신금융서비스]] 신청.

### Phase 3: 혁신금융서비스 신청 (2026-06 ~)

- **2026-04-27** 보험사 시간 끌기 전략 예상 → 명확한 대응 논리 준비 필요. 정부 자동차 보험료 인하 압박으로 금융위원회의 혁신금융서비스 신속 승인 가능성 높음 판단.
- 대응 방안: 4대 보험사 공통 항목 기준 데이터 분석, 외산차 20% / 국산차 80% 가중 평균 결과지 도출.
- **'더 낸 보험료 찾아드립니다'** 공격적 문구 → 보험사 요청으로 우호적 표현 검토.
- **결론**: 2026-06 [[혁신금융서비스]] 신청 전 보험사 협상 완료 목표. 운영파트 인력 부족 시 [[부품사업본부]] 인원 활용해 데이터 비교 작업 신속 처리.
- **2026-05-11** [[혁신금융서비스]] 신청 완료, 컨설팅 일정 조율 (2026.05.21~05.29).
- **2026-05-18** 네이버페이 미팅(2026.05 5주차 예정) 및 해빗팩토리 미팅 조율. 단독 추진보다 대형 플랫폼과 채널/수수료를 나누더라도 빠르게 진입하는 전략 검토.
