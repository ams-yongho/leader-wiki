# 클레임 모듈 (클라라/앨리스/아이리스/베라)

[[CA]] (Claims Agent) 확장으로서, 보험사 클레임 처리 자동화를 위한 AI/도메인 모듈군. 2026-04부터 본격 기획·개발.

## 모듈

### 클라라 (CLARA / Clara)

- **2026-04-13** 클레임 에이전트 프로젝트명은 **클라라 확정**. 내부 모듈 아이디어 구상 중.
- **2026-04-20** CLARA: 기획 문서 검증·작성 중, 향후 모듈 명칭으로 소통 예정.
- **2026-04-27** 스테이징 서버에서 시연 가능. 고객사 시연을 위해 데미지 디텍션 기능 추가 후 모바일 구현 검토. MVP 단계 기간계 연동 없이 기능 유효성 검증 방안 고려.

- **2026-04-27** Clara/Alice 모바일 웹 시연 가능성 검토 (액션 아이템).

- **2026-04-27** Inspektlabs 계약 체결 후 시연 가능토록 추가 개발 예정.

### 베라 (VERA)

- **2026-04-27** 스테이징 서버 시연 가능하나 목업 데이터 선정 필요.
