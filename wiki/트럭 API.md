# 트럭 API (상용차 운행점수 API)

상용차 운행 데이터를 분석해 안전운전 점수를 산출하고 보험사·운영사에 제공하는 API. **DTG(디지털 운행기록계) 통합**과 **아이나비 의존성 제거**가 핵심 트랙.

## 운영 통계

| 일자 | 접수 건수 |
|------|----------|
| 2026-02-23 | 2026.02.09~02.20 0건 (FTP 업데이트 없음, [[DB손해보험]] 측 확인 예정). 스카이오토넷 트럭 스코어 API 확장 검토. |
| 2026-03-16 | 2026.03.1주차 2건. |
| 2026-03-23 | 2026.03.21 누적 16건. |
| 2026-04-20 | 2026.04 3주차 6건. |
| 2026-04-27 | 2026.04 3주차 요청 3건 (상용차 운행점수 API). |
| 2026-05-04 | 2026.04 전체 32건. |

## 액션 아이템 (2026-04-27)

- 트럭 API 맵매칭 결과 → 아이리스 포맷 매칭 → 개발팀 전달 → 안전운전 점수 산출.
- DTG 스트레스 테스트 + 파일 기반 주행거리 정보 추가 + 릴리즈 도메인 작업.

## 한국보험금융 / 자이로드

- **2026-03-16** 한국보험금융 측서 DTG 업체로 자이로드 결정. 당사 위치정보사업자 등록 및 개발 기간 파악 필요.
- **2026-03-30** 지넥스 측에 위치정보사업자 등록 재요청.
- **2026-05-18** 한국보험금융 데이터 적재 테스트: 지넥스 측 적재 테스트 성공 확인 완료. 파일 구조는 기존 안전운전 점수 산출과 동일하게 설계했으나 실 점수 산출 정상 작동 여부 확인 필요.

## VIS 도입 (안전운전 특약 - 트럭 별도)

- **2026-05-04** VIS 도입 — 4대 보험사 안전운전 특약 제공 현황 확인 완료. 삼성화재 이슈: FCW(전방충돌경고)/AEB(자동비상제동) 명확 구분 불가, 케이스별 정리 후 보고 예정.
