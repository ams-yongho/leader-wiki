# raw/

리더회의 **원본 소스**를 보관합니다. 이 디렉토리는 **불변 영역**입니다 — LLM은 읽기만 하고 절대 수정·삭제하지 않습니다.

## 무엇을 넣나요

- 회의 녹음 전사본 (Whisper, 네이버 클로바, MS Teams 등)
- 수기 메모를 옮긴 마크다운/텍스트
- 회의 자료 PDF, 슬라이드, 스프레드시트
- 관련 이메일·메신저 캡처

## 파일 명명

권장 형식: `YYYY-MM-DD-주제.md`

예시:
- `2026-05-16-주간-리더회의.md`
- `2026-05-13-q2-okr-검토.md`

## 첨부 파일

이미지·녹음·PDF 등 바이너리·미디어는 `raw/assets/`에 보관하고, 위키 페이지나 회의록에서 상대경로로 참조하세요.

## 사용 흐름

1. 새 파일을 이 디렉토리에 넣는다.
2. LLM 에이전트에게 "방금 넣은 소스 ingest해줘"라고 요청한다.
3. LLM은 `wiki/` 페이지를 작성·갱신하고, `wiki/index.md`와 `wiki/log.md`를 업데이트한다.

자세한 워크플로우는 [../AGENTS.md](../AGENTS.md) 참고.
